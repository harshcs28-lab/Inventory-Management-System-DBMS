# Inventory-Management-System-DBMS
A complete DBMS project implementing an Inventory Management System using ER Modeling and SQL. The project includes database design, normalization up to 3NF, relational schema, and SQL queries (DDL, DML, Joins, Aggregate functions, Subqueries). Developed as part of academic coursework.
